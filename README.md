# word-bingo

WordBingoTask

# Requirement

- node v10.20.1 or above

# Installation

```bash
git init
git remote add origin https://github.com/kento-kuga/word-bingo.git
git pull origin master
npm install
```

# Usage

execute JavaScript

```bash
node dist/index.js
```

execute TypeScript

```bash
./node_modules/.bin/ts-node src/index.ts
```
